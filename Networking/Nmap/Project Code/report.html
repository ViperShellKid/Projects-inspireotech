Website Vulnerability Assessment

In-depth Security Infrastructure Analysis using Nmap

NAME: ARUN M KIT ID: 82375

Objective & Environment

Systematically scanning hotelvrisa.com to identify open ports, service versions, and SSL/TLS weaknesses using Nmap v7.98 on Kali Linux.

Initial Discovery & Port Scanning

Installation verification followed by host discovery and a standard TCP SYN scan of the top 1000 ports.
nmap -sn hotelvrisa.com
nmap hotelvrisa.com
Host Discovery: Alive
Exposed Ports: 21, 53, 80, 443, 1433, 3306

Service & OS Fingerprinting

Using aggressive scanning (-A) and version detection (-sV) to identify the specific software and operating system.
nmap -sV hotelvrisa.com
nmap -O hotelvrisa.com
OS: Windows Server
Web: Microsoft IIS 10.0
DB: Microsoft SQL Server 2022
Mail: MailEnable SMTP/POP3

Vulnerability & SSL Analysis

CRITICAL FINDINGS
Analyzing HTTP security headers and SSL/TLS cipher suites for cryptographic weaknesses.
nmap --script http-security-headers -p 80,443
nmap --script ssl-enum-ciphers -p 443
3DES Cipher Enabled (Sweet32 Vulnerable)
HSTS Enforcement Missing
Weak CBC Ciphers detected
Version Disclosure headers present

Key Vulnerabilities Identified

Excessive service exposure (FTP, SQL, Mail).
Cryptographic weakness (3DES Support).
Information Leakage via Service Headers.
Insecure Web Headers (Missing HSTS).

Strategic Recommendations

Disable 3DES and enforce TLS 1.2+.
Implement HSTS and hide IIS versions.
Close non-essential ports via Firewall.
Deploy IDS/IPS for real-time monitoring.
